# Emma - Apartment Website

A modern apartment showcase website built with HTML and Tailwind CSS.

## Overview

Emma is a responsive web application showcasing apartment listings and interior design. The site features a dark theme with a custom color palette optimized for modern apartment presentations.

## Project Structure

```
.
├── index.html          # Main HTML file
├── assets/             # Media assets
│   ├── Modern Interior Design For Apartment - Home Designing.mp4
│   └── Untitled_design__4_-removebg-preview.png
└── README.md          # This file
```

## Technologies Used

- **HTML5** - Markup structure
- **Tailwind CSS** - Utility-first CSS framework
- **Material Symbols** - Icon library
- **Google Fonts** - Custom typography (Noto Serif, Manrope)

## Features

- Responsive design with dark mode support
- Modern UI with custom color theme
- Clean, semantic HTML structure
- Tailwind CSS plugins included (forms, container queries)

## Getting Started

1. Open `index.html` in your web browser
2. The site will load with all styles and assets

## Customization

### Color Theme

The custom color palette can be modified in the Tailwind config section within `index.html`. Key colors include:
- Primary: `#b1cad7`
- Secondary: `#afcbd8`
- Tertiary: `#bbc8d1`
- Background: `#111415`

### Fonts

To change the font families, modify the Google Fonts link in the `<head>` section:
- **Primary font**: Noto Serif
- **Secondary font**: Manrope

## Browser Support

Works on all modern browsers that support HTML5 and CSS3.
